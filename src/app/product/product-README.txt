When you generate the Product Catalog schematic component, the following will be installed automatically:
1. npm install bootstrap@^4.6.0
2. npm install @angular/material@^11.2.13
3. npm install @angular/cdk@^11.2.13
4. npm install @angular/animations@^11.2.14
7. npm install @angular/flex-layout

NOTE: you need to add the Bootstrap in the - in the src/styles.css with the following code:

    @import "../node_modules/bootstrap/dist/css/bootstrap.css";
